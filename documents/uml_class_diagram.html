Class Relationships

  • User: Base class for Students and Instructors with authentication methods
  • Course: Course management with content and assignments
  • Enrollment: Junction class for Many-to-Many relationship between User and Course
  • CourseContent: Manages PDF files and course materials
  • Assignment: Creates assignments within courses
  • Submission: Student submissions with timestamps
  • Grade: Grading and feedback on submissions
